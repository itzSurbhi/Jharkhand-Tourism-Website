# 🌄 Jharkhand Tourism Website (Frontend Only)

Welcome to the official frontend project for the **Jharkhand Tourism Website**! This project showcases the natural beauty, rich culture, historical landmarks, and travel information for the Indian state of Jharkhand. It's designed to provide a clean, responsive, and visually appealing interface for tourists and travelers.

---

## 🖼️ Features

- ✅ Fully responsive design (Mobile, Tablet, Desktop)
- ✅ Home page with hero section, highlights, and call to action
- ✅ Explore Places: Detailed cards with images and descriptions
- ✅ Cultural Heritage: Information on festivals, dance, and art
- ✅ Travel Guide: Tips, weather, best time to visit
- ✅ Gallery: Image carousel/slideshow
- ✅ Contact Us form (UI only – no backend integration)
- ✅ Footer with quick links and social media icons

---
